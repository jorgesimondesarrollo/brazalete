# Plantilla básica ASP.NET Core + Blazor Server

Esta plantilla crea una app Blazor Server mínima para arrancar rápido.

## Requisitos
- .NET SDK 8 (o superior)

## Ejecutar
```bash
dotnet run --project BlazorApp
```

Luego abre `http://localhost:5000`.

## Estructura
- `BlazorApp/Program.cs`: configuración de la app.
- `BlazorApp/Pages`: páginas Razor.
- `BlazorApp/Shared`: layout y navegación.
- `BlazorApp/wwwroot`: recursos estáticos.
